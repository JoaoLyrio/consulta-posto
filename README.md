Monitoramento de Combustíveis - Vila Velha

Este projeto tem como objetivo monitorar e analisar os preços de combustíveis em postos da região de Vila Velha - ES.

Funcionalidades

- Identificação do menor e maior preço
- Cálculo da média de preços por posto
- Consulta do preço mais recente
- Histórico de variação de preços

Objetivo

Ajudar a comunidade a economizar ao identificar os postos com melhores preços.

Estrutura do Banco

O banco de dados é composto pelas seguintes tabelas:

- postos
- telefones_posto
- tipos_combustivel
- coletas
